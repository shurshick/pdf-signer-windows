# PDF Signer Windows v0.2.0

## Русский

Существенный релиз Windows-версии PDF Signer с поддержкой открепленной подписи `.sig`, улучшенным отбором сертификатов и обновленным штампом.

### Что добавлено

- Чекбокс для создания открепленной CAdES-подписи в формате `.sig`.
- При включенной опции рядом с результатом создается отдельный `.sig` файл.
- На штамп добавлена ключевая проверочная информация: SHA-256 хэш данных, SHA-256 хэш `.sig` при создании открепленной подписи и отпечаток сертификата.
- Штамп теперь отображается синим цветом.

### Что изменилось

- Из списка доступных сертификатов убраны истекшие и еще не действующие сертификаты.
- Версия приложения обновлена до `0.2.0`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступным CAdESCOM.
- Действующий сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.2.0-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.2.0-portable.zip` - portable-версия без установки.

### Важно

Сборка проверена в GitHub Actions. Реальное подписание и создание `.sig` нужно проверять на Windows-машине с установленным CryptoPro CSP и рабочим сертификатом.

## English

Major update of the Windows version of PDF Signer with detached `.sig` support, improved certificate filtering, and an updated stamp.

### Added

- Checkbox for creating a detached CAdES `.sig` signature.
- When enabled, the app creates a separate `.sig` file together with the signed output.
- The stamp now includes key verification information: SHA-256 data hash, SHA-256 `.sig` hash when a detached signature is created, and certificate thumbprint.
- The stamp is now rendered in blue.

### Changed

- Expired and not-yet-valid certificates are hidden from the selectable certificate list.
- Application version updated to `0.2.0`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM available.
- A currently valid signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.2.0-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.2.0-portable.zip` - portable build without installation.

### Important

The build is verified by GitHub Actions. Real signing and `.sig` creation must be tested on a Windows machine with CryptoPro CSP and a working certificate.
