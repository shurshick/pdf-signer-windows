# PDF Signer Windows v0.3.0

## Русский

Существенный релиз с обновленной логикой открепленной подписи и выбором папки сохранения.

### Что добавлено

- Новый чекбокс для сохранения результата рядом с исходным PDF.
- Если чекбокс не отмечен, файлы по-прежнему сохраняются в указанную папку результата.

### Что изменилось

- Режим открепленной подписи `.sig` теперь не встраивает подпись в PDF.
- При включенной открепленной подписи приложение создает `.sig` и PDF-копию только с видимым штампом.
- Обычный режим без `.sig` продолжает встраивать подпись в PDF и добавлять штамп.
- Версия приложения обновлена до `0.3.0`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступным CAdESCOM.
- Действующий сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.3.0-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.3.0-portable.zip` - portable-версия без установки.

## English

Major release with updated detached-signature behavior and output location selection.

### Added

- New checkbox for saving outputs next to the source PDF.
- If the checkbox is not selected, files are still saved to the selected output folder.

### Changed

- Detached `.sig` mode no longer embeds a signature into the PDF.
- When detached signing is enabled, the app creates a `.sig` file and a PDF copy with only the visible stamp.
- Regular mode without `.sig` still embeds the signature into the PDF and adds the visible stamp.
- Application version updated to `0.3.0`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM available.
- A currently valid signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.3.0-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.3.0-portable.zip` - portable build without installation.
