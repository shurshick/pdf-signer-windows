# PDF Signer Windows v0.2.2

## Русский

Исправительный релиз для машин, где `cryptcp.exe` не находится автоматически.

### Что исправлено

- Добавлено поле `cryptcp.exe` в интерфейсе.
- Приложение пытается найти `cryptcp.exe` через PATH, типовые папки CryptoPro, системные папки и записи реестра.
- Если автопоиск не помог, пользователь может выбрать `cryptcp.exe` вручную.
- Выбранный путь сохраняется в профиле пользователя.
- Версия приложения обновлена до `0.2.2`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступными CAdESCOM и `cryptcp.exe`.
- Действующий сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.2.2-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.2.2-portable.zip` - portable-версия без установки.

## English

Bugfix release for machines where `cryptcp.exe` is not auto-detected.

### Fixed

- Added a `cryptcp.exe` field to the UI.
- The app now searches PATH, common CryptoPro folders, system folders, and registry locations.
- If auto-detection is not enough, users can choose `cryptcp.exe` manually.
- The selected path is saved in the user profile.
- Application version updated to `0.2.2`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM and `cryptcp.exe` available.
- A currently valid signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.2.2-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.2.2-portable.zip` - portable build without installation.
