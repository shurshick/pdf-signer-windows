# PDF Signer Windows v0.2.1

## Русский

Исправительный релиз для формирования открепленной подписи `.sig`.

### Что исправлено

- Открепленная подпись `.sig` теперь создается через CryptoPro `cryptcp.exe` в режиме `-sign -der -detached`.
- Приложение больше не записывает внешний `.sig` напрямую из байтов CAdESCOM/PKCS#7.
- Если `cryptcp.exe` не найден или не смог создать подпись, приложение показывает ошибку вместо создания файла неподходящего формата.
- Версия приложения обновлена до `0.2.1`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступными CAdESCOM и `cryptcp.exe`.
- Действующий сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.2.1-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.2.1-portable.zip` - portable-версия без установки.

## English

Bugfix release for detached `.sig` signature generation.

### Fixed

- Detached `.sig` signatures are now created through CryptoPro `cryptcp.exe` with `-sign -der -detached`.
- The app no longer writes the external `.sig` directly from CAdESCOM/PKCS#7 bytes.
- If `cryptcp.exe` is missing or fails to create the signature, the app shows an error instead of producing a file in the wrong format.
- Application version updated to `0.2.1`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM and `cryptcp.exe` available.
- A currently valid signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.2.1-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.2.1-portable.zip` - portable build without installation.
