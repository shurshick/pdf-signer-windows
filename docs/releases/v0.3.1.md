# PDF Signer Windows v0.3.1

## Русский

Исправительный релиз для логики открепленной подписи и содержимого штампа.

### Что исправлено

- В режиме открепленной подписи `.sig` приложение сначала создает PDF со штампом, а затем формирует `.sig` именно от этого проштампованного PDF.
- Файл `.sig` теперь относится к тому PDF, который пользователь видит и передает дальше.
- Со штампа убраны хэши данных и `.sig`, чтобы не создавать циклическую зависимость и не вводить пользователя в заблуждение.
- На штампе остаются стабильные сведения: подписант, дата, назначение и отпечаток сертификата.
- Версия приложения обновлена до `0.3.1`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступным CAdESCOM.
- Действующий сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.3.1-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.3.1-portable.zip` - portable-версия без установки.

## English

Bugfix release for detached-signature behavior and stamp contents.

### Fixed

- In detached `.sig` mode, the app now creates the stamped PDF first, then creates the `.sig` for that stamped PDF.
- The `.sig` file now belongs to the exact PDF that the user sees and shares.
- Data and `.sig` hashes were removed from the visible stamp to avoid circular dependencies and misleading verification cues.
- The stamp keeps stable information: signer, date, reason, and certificate thumbprint.
- Application version updated to `0.3.1`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM available.
- A currently valid signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.3.1-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.3.1-portable.zip` - portable build without installation.
