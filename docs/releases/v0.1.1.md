# PDF Signer Windows v0.1.1

## Русский

Исправительный релиз Windows-версии PDF Signer.

### Что изменилось

- Видимый штамп подписи теперь размещается на всех страницах PDF, а не только на первой.
- Штампы добавляются в содержимое PDF до криптографической подписи, поэтому они входят в подписываемые данные.
- Версия приложения обновлена до `0.1.1`.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступным CAdESCOM.
- Сертификат подписи с доступным закрытым ключом в хранилище Windows.

### Что скачать

- `pdf-signer-windows-0.1.1-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.1.1-portable.zip` - portable-версия без установки.

### Важно

Сборка проверена в GitHub Actions. Реальное подписание нужно проверять на Windows-машине с установленным CryptoPro CSP и рабочим сертификатом.

## English

Bugfix release of the Windows version of PDF Signer.

### Changed

- The visible signature stamp is now placed on every PDF page instead of only the first page.
- Stamps are added to the PDF content before the cryptographic signature, so they are covered by the signed data.
- Application version updated to `0.1.1`.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM available.
- A signing certificate with an accessible private key in the Windows certificate store.

### Downloads

- `pdf-signer-windows-0.1.1-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.1.1-portable.zip` - portable build without installation.

### Important

The build is verified by GitHub Actions. Real signing must be tested on a Windows machine with CryptoPro CSP and a working certificate.
