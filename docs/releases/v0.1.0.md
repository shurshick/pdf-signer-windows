# PDF Signer Windows v0.1.0

## Русский

Первый MVP-релиз Windows-версии PDF Signer. Это отдельный проект для Windows 7 SP1, Windows 8.1, Windows 10 и Windows 11. Он не заменяет Linux-репозиторий `shurshick/pdf-signer`.

### Что добавлено

- Выбор одного или нескольких PDF-файлов.
- Выбор сертификата из хранилищ Windows `CurrentUser\My` и `LocalMachine\My`.
- Подписание PDF через CryptoPro CAdESCOM.
- Встраиваемая отсоединенная CAdES-подпись в PDF.
- Видимый штамп подписи на первой странице.
- Пакетное подписание нескольких PDF за один запуск.
- Автоматический русский или английский интерфейс по системному языку.
- GitHub Actions сборка для portable ZIP и установщика NSIS.

### Требования

- Windows 7 SP1, Windows 8.1, Windows 10 или Windows 11.
- .NET Framework 4.8.
- Установленный CryptoPro CSP с доступным CAdESCOM.
- Сертификат подписи с доступным закрытым ключом в хранилище Windows.

CryptoPro CSP, сертификаты и ключевые контейнеры не входят в поставку приложения.

### Что скачать

- `pdf-signer-windows-0.1.0-setup.exe` - установщик для обычной установки в профиль пользователя.
- `pdf-signer-windows-0.1.0-portable.zip` - portable-версия без установки.

### Важно

Сборка проверена в GitHub Actions. Реальное подписание нужно проверять на Windows-машине с установленным CryptoPro CSP и рабочим сертификатом, потому что CAdESCOM и закрытые ключи доступны только в такой среде.

## English

First MVP release of the Windows version of PDF Signer. This is a separate project for Windows 7 SP1, Windows 8.1, Windows 10, and Windows 11. It does not replace the Linux repository `shurshick/pdf-signer`.

### Added

- Select one or more PDF files.
- Select a certificate from Windows `CurrentUser\My` and `LocalMachine\My` stores.
- Sign PDFs through CryptoPro CAdESCOM.
- Embed a detached CAdES signature into the PDF.
- Add a visible signature stamp on the first page.
- Batch-sign multiple PDFs in one run.
- Automatically choose Russian or English UI from the system language.
- GitHub Actions build for a portable ZIP and an NSIS installer.

### Requirements

- Windows 7 SP1, Windows 8.1, Windows 10, or Windows 11.
- .NET Framework 4.8.
- CryptoPro CSP installed with CAdESCOM available.
- A signing certificate with an accessible private key in the Windows certificate store.

CryptoPro CSP, certificates, and key containers are not bundled with the application.

### Downloads

- `pdf-signer-windows-0.1.0-setup.exe` - installer for a regular per-user installation.
- `pdf-signer-windows-0.1.0-portable.zip` - portable build without installation.

### Important

The build is verified by GitHub Actions. Real signing must be tested on a Windows machine with CryptoPro CSP and a working certificate, because CAdESCOM and private keys are only available in that environment.
